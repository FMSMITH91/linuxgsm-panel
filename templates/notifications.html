Notifications

Get alerted on Telegram or Discord when something needs your attention — a server drops, a host goes unreachable, a backup fails, an admin signs in. This is the panel notifying you; a game's own in-game alerts are still set per server on its Alerts tab.

Telegram
Create a bot with @BotFather, then get your chat ID from @userinfobot (or a group's ID). Message the bot once so it can reply to you.
Lets you run the panel from Telegram — only messages from the Chat ID above are honoured. Send /help in the chat for the list.
If that Chat ID is a group, every member can run these commands — including /start, /stop, and /restart. Use a private chat for server control.
Discord
In your Discord channel: Edit Channel → Integrations → Webhooks → New Webhook → Copy URL. Only discord.com webhook URLs are accepted.
Two-way command bot (optional)
Create a bot at discord.com/developers → New Application → Bot, copy its token, and invite it to your server. Under Bot → Privileged Gateway Intents turn on Message Content Intent so it can read commands. Get the Channel ID with User Settings → Advanced → Developer Mode, then right-click the channel → Copy Channel ID. Type !help in that channel for the command list (Discord reserves / for its own menu, so ! types cleanly).
Everyone who can post in that channel can run these commands — including !start, !stop, and !restart. Use a private, admin-only channel for server control.
