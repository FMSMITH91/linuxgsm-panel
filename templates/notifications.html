Notifications

Get alerted on Telegram or Discord when something needs your attention — a server drops, a host goes unreachable, a backup fails, an admin signs in. This is the panel notifying you; a game's own in-game alerts are still set per server on its Alerts tab.

Telegram
Create a bot with @BotFather, then get your chat ID from @userinfobot (or a group's ID). Message the bot once so it can reply to you.
Discord
In your Discord channel: Edit Channel → Integrations → Webhooks → New Webhook → Copy URL. Only discord.com webhook URLs are accepted.
