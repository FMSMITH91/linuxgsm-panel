Welcome to LinuxGSM Panel!

This wizard will guide you through the initial setup.
You'll set up your site and create an admin account. After that you can install game servers right here on this machine, or on remote VPS nodes over SSH.

What you'll need:
  • A Linux host (Ubuntu recommended) — this machine and/or a remote VPS. LinuxGSM and the game servers are installed for you — nothing to pre-install.
  • For a remote host: SSH access (key, password, or Tailscale SSH).
  • A sudo-capable user (or root) on whichever host you install game servers on.
Use 127.0.0.1 behind a reverse proxy, 0.0.0.0 for direct access
